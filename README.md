## 구현 기능
- 회원가입 트랜잭션을 이용하여 구현
- swagger 를 활용해 Api 문서화 작성
- passport 및 Guarud 을 만들어 회원인증 구현
- class-validator 이용하여 유효성 검사구현
- 유닛테스트(unit test)를 통해 service 로직 테스트
- socket.io 를 이용하여 채팅 구현
- 채팅에 image 업로드 구현
